### Repeater
Repeats text with pattern

![Repeater](https://github.com/amenayach/Repeater/blob/master/doc/screenshot.png?raw=true)

So result will be like:
```javascript
{
	"id": 0,
	"name": "Someone0",
	"good": 0,
	"date": new Date(0, 1, 1),
	"template": "aaa"
},
{
	"id": 1,
	"name": "Someone1",
	"good": 1,
	"date": new Date(1, 1, 1),
	"template": "bbb"
},
{
	"id": 2,
	"name": "Someone2",
	"good": 0,
	"date": new Date(2, 1, 1),
	"template": "ccc"
},
```
